HTML base

Atributos

Atributos contem informações adicionais sobre elementos. Todos os atributos são compostos de nome e valor.

  • Todos elementos html podem ter atributos.

  • Atributos são sempre especificados na tag inicial.

  • Atributos geralmente aparecem em pares de nome/valor

Cabeçalhos

  • Cabeçalhos são definidos da tag h1 a tag h6

  • h1 define o cabeçalho mais importante. h6 define o cabeçalho menos importante.

  • A tag só pode e deve ser usada uma única vez, as demais não há limites de uso.

Paragrafos

  • O parágrafo é o elemento base no texto editado. A tag para um parágrafo é p

  • O HTML adiciona automaticamente uma linha em branco extra antes e após um parágrafo.

  • Nem sempre um parágrafo é um conjunto de linhas. Existem parágrafos com somente uma palavra ou uma linha.
