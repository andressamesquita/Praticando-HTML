Materialize

O que é o Materialize?

Materialize é um framework CSS desenvolvido pela google com base no material design.

Com seus componentes bem semelhantes ao do Bootstrap, ele apresenta opções com design mais leve. Isto faz com que seu projeto fique visualmente mais agradável.

Possui uma documentação vasta de fácil entendimento.

Como Começar a Utilizar o Materialize?

O materialize pode ser instalado de 3 formas:

  • Online: link para um site

    • <link rel="stylesheet" href="https://cdnjs.cloudflare.com/
    ajax/libs/materialize/1.0.0/css/materialize.min.css>

  • Offline: link direto para o arquivo

    • <link rel="stylesheet" href="/css/materialize.min.css>

  • SASS: Um pré-processador de CSS
