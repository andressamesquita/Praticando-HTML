Bootstrap

O que é o Bootstrap?

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

Foi desenvolvido por Jacob Thorton e Mark Otto, engenheiros do Twitter, como uma tentativa de resolver incompatibilidades dentro da própria equipe

Em que ele pode ser utilizado?

Na prática, a principal aplicação do Bootstrap seria na criação de sites responsivos. Com o bootstrap o desenvolvedor não perde tempo criando todo uma linha de CSS novamente, devido ao fato de que ele possui diversos plugins em JavaScript(jQuery).

Características e vantagens do Bootstrap:

  • Biblioteca de componentes

  • Reuso de código

  • Documentação e comunidade

  • Padrão visual

  • Responsividade

Como usar o Bootstrap?

Fazendo o download do Bootstrap no site oficial da ferramenta e adicionando os arquivos "bootstrap.min.css" e "bootstrap.min.js" dentro das pasta correspondente no seu site. Depois, chamar os arquivos na tag <head> do HTML.

O Bootstrap utiliza a biblioteca jQuery, então é importante inseri-lo na página para que tudo funcione corretamente.

Exemplo:

<link rel="stylesheet" href="/css/bootstrap.min.css"/>
<script src="js/bootstrap.min.js">
<script type="text/javascript" src="/javascript/jquery/jquery-3.3.1.min.js"><script/>
